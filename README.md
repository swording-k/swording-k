# Skyler.Yang

I build small, opinionated software experiments around AI, personal tools, and interactive products.

Most of my work sits somewhere between product design and engineering: taking an idea, shaping it into a usable interface, and pushing it far enough that someone can actually try it.

## Featured Work

- [Fit Genius](https://github.com/swording-k/fit-genius) - an AI-assisted iOS fitness planner for turning goals, routines, and progress into a more adaptive training experience.
- [The Last Bookstore](https://github.com/swording-k/the-last-bookstore) - a browser-playable narrative AI project about memory, books, and a digital world after collapse.
